============
Introduction
------------

The Minisip Library Stack Working Group seeks to develop and maintain the
core code modules and libraries, providing support to the Application WG 
and other groups as necessary to deliver a set of generic, stable components
useful for building interactive collabaration services.

=========
Libraries
---------

This working group oversees the development of the following libraries:

 * libmutil
 * libmcrypto
 * libmnetutil
 * libmikey
 * libmsip
 * libminisip

The following plug-in modules are also managed by this WG:
 * milbc
