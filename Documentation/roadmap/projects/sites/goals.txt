Introduction
------------

This document presents the long-term goals for the Sites Working Group.
These serve to guide the group on a clear path for fulfilling the group's 
specific functions within The Project.

===========
Group Goals
-----------

1) "First impressions last a lifetime, so ours should be good."
  a) The web site will be the first impression of our project, so it
     should use multiple servers on various networks (with high uptimes).
  b) Our test suite may require a public test server, which must perform
     similarly when users decide to try it.

2) "Organization may be temporary, URLs can be forever."
  a) All release URLS must be checked periodically for integrity.
  b) Paths that require encoding should be avoided for readability.
