============
Introduction
------------

The Internationalization (i18n) Working Group seeks to develop and maintain
the translations of all Project software and documentation into the various
languages required by our users.

=========
Languages
---------

The I18N Working Group currently targets the following languages:

 * English

In fact, this Working Group was established as a means of organizing the
resources necessary to begin the process of adding the required support.

In other words, this WG has yet to begin its work.

=============
Project Files
-------------

This Working Group actively maintains the following files in this directory:

 * README.txt		- this file
 * goals.txt		- detailed project working group goals
 * process.txt		- this working group' policies and procedures

Other files may exist but should be referenced after the preceeding files.
